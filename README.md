# python-cli-bazel-pylogging-pascal-functional-prog

## Description
Find the binomial coefficent.
Follows functional programming practices
POC for logging.

## Tech stack
- python

## Docker stack
- l.gcr.io/google/bazel:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
